# CodeAlpha Object Detection and Tracking

## Description
This project is an Object Detection and Tracking system developed using Python, OpenCV, and YOLOv8 as part of the CodeAlpha Artificial Intelligence Internship.

## Features
- Real-time object detection
- Object tracking with unique IDs
- Webcam support
- Bounding boxes with labels
- Confidence score display

## Technologies Used
- Python 3
- OpenCV
- Ultralytics YOLOv8

## How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
